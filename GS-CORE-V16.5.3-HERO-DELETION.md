# GS CORE V16.5.3 — Hero Deletion

Добавлена безопасная команда `/hero delete`.

- Два последовательных подтверждения кнопками.
- Кнопками может пользоваться только владелец героя.
- Удаляется герой и весь относящийся к нему RPG-прогресс.
- Сохраняются карточки, Dust, достижения сообщества, ежедневная активность и основной профиль участника.
- После удаления игрок может сразу использовать `/hero create`.
- Серверные первые убийства мини-боссов и общий прогресс мира сохраняются.
