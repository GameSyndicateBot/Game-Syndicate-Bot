# GS CORE STABLE V15.6.2 — Crafting & Blacksmith

Добавлено:
- `/craft list [available]` — книга рецептов;
- `/craft info recipe:` — состав и доступность рецепта;
- `/craft create recipe: [quantity:]` — создание предметов;
- 24 рецепта кузнеца и алхимика;
- требования по уровню героя;
- атомарное списание материалов и Dust;
- история крафта в `hero_crafting_history`;
- автодополнение рецептов с отметкой доступности.

После обновления зарегистрировать команды:
`npm run deploy:prod`
