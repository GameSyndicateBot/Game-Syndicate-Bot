# GS CORE STABLE V15.5 — Hero × World Boss

- World Boss registration now requires a hero created through the expedition/RPG system.
- The hero's chosen name is shown on the battle card, team list, turn queue, rolls and personal status.
- Discord mention remains visible as the owner of the hero.
- Hero level, equipped items and active companion are snapshotted when joining the raid.
- The raid class is still temporary and selected using the original d20 class-order flow.
- Hero progression is capped: up to +25% outgoing basic damage, +20% class HP and +15% resistance.
- Existing World Boss class skills, initiative and cooperative roles remain the primary source of combat power.
- Existing battles and databases migrate automatically with additive columns only.
