# GS CORE V16.5.4 — Interactive Guide

## Добавлено

- Новая slash-команда `/guide`.
- 13 разделов встроенного руководства.
- Навигация через выпадающее меню и кнопки «Назад», «Главная», «Далее».
- Возможность сразу открыть раздел через параметр `/guide section`.
- Кнопка «Открыть руководство» после создания нового героя.
- Руководство показывается приватно и не засоряет игровой канал.

## Разделы

- Герой
- Классы
- Экспедиции
- Экипировка
- Материалы и сундуки
- Алхимия
- Крафт и улучшение
- Мини-боссы
- Живой мир
- Гильдия героев
- Мировой Босс
- Dust, магазин и карточки
- Частые вопросы

## После установки

Обновить slash-команды:

```bash
npm run deploy:prod
```
