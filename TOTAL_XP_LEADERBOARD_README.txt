TOTAL XP LEADERBOARD FIX

Что было:
XP-лидерборд сортировал игроков по полю players.xp.
Это только текущий XP внутри нынешнего уровня, поэтому игрок после повышения уровня
мог внезапно опуститься в топе.

Что стало:
Общий XP считается так:
- сумма XP, необходимого для всех уже пройденных уровней;
- плюс текущий XP внутри нынешнего уровня.

Пример:
Игрок уровня 10 с 500 текущего XP будет выше игрока уровня 5 с 900 текущего XP,
потому что учитывается весь опыт за всё время.

Изменено:
- сортировка XP-лидерборда;
- значение XP, отображаемое на карточке топа.

Установка:
1. Остановить бота: Ctrl + C.
2. Распаковать ZIP в корень GS-Core с заменой файлов.
3. Запустить: node index.js

deploy-commands.js запускать не нужно.
