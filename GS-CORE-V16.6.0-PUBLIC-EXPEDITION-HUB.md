# GS CORE V16.6.0 — Public Expedition Hub

## Добавлено
- Кнопка администратора **«Пересоздать хаб экспедиций»** в `/setup` и `/control`.
- Старые сообщения Expedition Hub от бота удаляются.
- Новый хаб создаётся обычным публичным сообщением через `channel.send()`.
- Новый ID сохраняется в runtime и используется последующими автообновлениями.
- В лог выводится ID нового сообщения и число удалённых старых хабов.

## Использование
1. Установить обновление и перезапустить бота.
2. Выполнить `/setup` или `/control`.
3. Нажать **«Пересоздать хаб экспедиций»**.
4. Проверить видимость сообщения обычным участником.

Новых slash-команд нет, повторная регистрация команд не требуется.
