GS TELEGRAM — GATHER ONLY

После установки Telegram-бот оставляет только:
- /gather — создание игрового сбора;
- /cancel — отмена пошагового создания;
- кнопки Иду / Не смогу / Напомнить / Запустить / Отменить;
- синхронизацию с назначенным Discord-каналом;
- напоминания за 30 и 10 минут и в момент старта.

Удалено из Telegram-интерфейса:
- привязка Telegram к Discord;
- профиль;
- Dust;
- карты;
- достижения;
- настройки;
- главное мобильное меню.

Discord-команда /linktelegram отключена.

Установка:
1. Остановить бота: Ctrl + C.
2. Распаковать ZIP в корень GS-Core с заменой файлов.
3. Выполнить: node deploy-commands.js
   Это удалит /linktelegram из списка команд Discord.
4. Запустить: node index.js

Настройка Discord-канала сборов сохраняется через /setgatherchannel.
