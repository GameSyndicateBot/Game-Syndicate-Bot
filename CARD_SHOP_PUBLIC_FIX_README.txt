CARD SHOP PUBLIC ACCESS FIX

Причина:
Команда /cardshop и её кнопки были защищены checkDeveloper().
Из-за этого обычные участники могли видеть магазин, но покупка паков не работала.

Исправлено:
- /cardshop доступен всем участникам;
- кнопки Base, Premium и Elite Pack доступны всем;
- проверка баланса Dust сохранена;
- административные команды не затронуты.

Установка:
1. Остановить бота: Ctrl + C.
2. Распаковать ZIP в корень GS-Core с заменой файла.
3. Запустить: node index.js

deploy-commands.js запускать не нужно, структура slash-команды не менялась.
