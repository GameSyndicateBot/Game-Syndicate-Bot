# Game Syndicate Bot — v22 Safe Update Packaging

## Главное изменение

Закреплён безопасный формат обновлений рабочего бота: новый ZIP распаковывается **поверх текущей папки** с заменой файлов. Искать, копировать или восстанавливать `.env` и базы вручную не требуется.

## Добавлено

- `scripts/update-safety-audit.js` — проверяет, что update ZIP не содержит рабочий `.env` и SQLite-базы.
- `npm run update:audit` — отдельная проверка безопасности обновления.
- `npm run preflight` — запускает синтаксическую проверку, update-аудит и коммерческий аудит одной командой.
- `UPDATE-INSTRUCTIONS.md` — единая инструкция без ручного поиска файлов.

## Защита данных

Update ZIP намеренно не содержит:

- `.env`;
- `database/database.sqlite`;
- `telegram/data/telegram-gatherings.sqlite`;
- рабочие резервные копии.

При распаковке поверх существующей папки эти файлы остаются нетронутыми, а код и ресурсы обновляются.

## Совместимость

- Структура базы данных не менялась.
- Миграции не требуются.
- Игровые механики не менялись.
- Статус: Production Safe при установке поверх существующей папки.
