GS СБОРЫ v3 — СНАЧАЛА ОПОВЕЩЕНИЕ, ПОТОМ ЗАПИСЬ

Логика:
1. Организатор создаёт сбор.
2. Discord сразу отправляет @everyone.
3. Telegram отправляет громкое сообщение и временно закрепляет его с уведомлением.
4. Участники нажимают:
   - Иду
   - Не смогу
   - Напомнить
5. За 30 и 10 минут уведомляются:
   - записавшиеся;
   - попросившие напомнить.
6. В момент старта уведомляются те же люди.

Команда Telegram:
- /сбор
- /gather оставлена как запасная.

Важно:
Telegram не поддерживает настоящий @everyone.
Ближайший официальный вариант — закрепление сообщения с уведомлением.
Для этого боту нужны права администратора на закрепление сообщений.

Установка:
1. Ctrl + C
2. Распаковать поверх ранее установленного v2
3. node deploy-commands.js
4. node index.js
