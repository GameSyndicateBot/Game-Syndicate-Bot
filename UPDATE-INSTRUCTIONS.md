# Обновление рабочего Game Syndicate Bot

Эта ZIP-сборка предназначена для обновления действующего бота **поверх существующей папки**.

## Что делает владелец бота

1. Останавливает процесс бота.
2. Открывает ZIP и распаковывает его в текущую папку бота.
3. Соглашается на замену существующих файлов.
4. Запускает бота.

Удалять старую папку перед обновлением нельзя: рабочие данные специально не входят в update ZIP и остаются в существующей папке.

## Что сборка не перезаписывает

- `.env` — токены и настройки сервера;
- `database/database.sqlite` — профили, XP, карточки, экономика, достижения и другая Discord-статистика;
- `telegram/data/telegram-gatherings.sqlite` — данные Telegram;
- существующие каталоги резервных копий.

## Автоматическая проверка перед публикацией

```bash
npm run preflight
```

Проверка завершается ошибкой, если в update-сборку случайно попали `.env` или рабочие SQLite-базы.

## Важно

Update ZIP и коммерческий архив для покупателя — разные сборки. Коммерческий архив собирается отдельно и не содержит данных текущего сервера.
