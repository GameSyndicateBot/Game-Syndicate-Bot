# GS World Boss V10 — типы входящего урона

## Сопротивления классов
Каждый класс получил отдельное сопротивление физическому и магическому урону. Положительное значение уменьшает входящий урон, отрицательное означает уязвимость.

- Воин: физ. +18%, маг. -8%
- Паладин: физ. +12%, маг. +10%
- Страж: физ. +25%, маг. -12%
- Клирик: физ. -12%, маг. +12%
- Жрец: физ. -15%, маг. +15%
- Бард: физ. -8%, маг. +8%
- Ассасин: физ. +3%, маг. -5%
- Лучник: физ. +5%, маг. -6%
- Маг: физ. -18%, маг. +20%
- Берсерк: физ. +8%, маг. -10%
- Инженер: физ. +10%, маг. -5%
- Некромант: физ. -10%, маг. +18%

## Типы атак боссов
Каждый босс имеет собственный профиль атак и может выбирать физический или магический тип при обычной, особой, массовой и яростной атаке.

- Теневой Страж: 45% физический / 55% магический
- Пожиратель Пустоты: 25% / 75%
- Архонт Хаоса: 50% / 50%
- Железный Колосс: 85% / 15%
- Грозовой Тиран: 20% / 80%
- Костяной Император: 65% / 35%
- Ледяной Левиафан: 40% / 60%
- Багровый Дракон: 60% / 40%

Миньоны также получили фиксированный тип урона, соответствующий их тематике.

Журнал боя теперь показывает тип каждой атаки. В личной карточке класса показываются оба сопротивления.
