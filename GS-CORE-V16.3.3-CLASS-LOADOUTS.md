# Game Syndicate V16.3.3 — Class Loadouts

- Отдельный комплект экипировки для каждого из 12 классов.
- Общая экипировка остаётся запасным комплектом.
- World Boss автоматически использует комплект выбранного класса.
- `/hero equipment class:<класс>` показывает классовый комплект.
- `/hero equip id:<ID> class:<класс>` сохраняет предмет в комплект класса.
- `/hero unequip slot:<слот> class:<класс>` снимает предмет только из выбранного комплекта.
- Ограничения бонусов World Boss из V16.3.2 сохранены.
