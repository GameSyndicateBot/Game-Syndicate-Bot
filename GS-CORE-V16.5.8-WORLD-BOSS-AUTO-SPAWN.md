# GS CORE V16.5.8 — WORLD BOSS AUTO SPAWN

## Изменено
- Автоматический спавн Мирового Босса включён по умолчанию.
- Расписание: 13:00 и 20:00 по Москве.
- После перезапуска активная регистрация или бой восстанавливаются как раньше.
- Ручной запуск администратора остаётся доступен.

## Управление через окружение
- `WORLD_BOSS_AUTO_SCHEDULE=false` — временно отключить автоспавн.
- При отсутствии переменной автоспавн включён.

## Установка
Достаточно заменить файлы и перезапустить бота. Повторный deploy slash-команд не нужен.
