# GS CORE V16.0.1 — UI ICON FIX

Исправлено отображение битых emoji-глифов в PNG-интерфейсах.

## Гильдия героев
- Удалены Unicode emoji из генератора главного хаба.
- Добавлены рисуемые Canvas-иконки для героя, профиля, инвентаря, кузнеца,
  алхимика, питомцев, артефактов, кодекса и экспедиций.

## World Boss
- Удалены emoji из ярости, миньонов, статуса, эффектов и нижней легенды.
- Добавлены рисуемые Canvas-иконки HP, маны, энергии, ярости, ульты,
  миньонов и статуса боя.
- Emoji из строк журнала боя очищаются перед отрисовкой, поэтому хостинг
  больше не выводит квадраты с кодами Unicode.
