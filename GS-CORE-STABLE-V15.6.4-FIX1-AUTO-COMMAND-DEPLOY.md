# GS CORE STABLE V15.6.4 FIX 1

## Исправление

Bothost Web Terminal не поддерживает запуск `npm` и `node` вручную. Поэтому регистрация slash-команд теперь автоматически запускается внутри контейнера перед стартом бота.

При каждом обновлении/перезапуске в логах должны появиться строки:

```text
🚀 Автоматическая регистрация Discord slash-команд...
🚀 Регистрация ... уникальных команд: основной сервер...
✅ Один комплект команд зарегистрирован на сервере ...
✅ Discord slash-команды зарегистрированы автоматически
```

Для регистрации необходимы переменные окружения:

- `TOKEN`
- `CLIENT_ID`
- `PROD_GUILD_ID` или `GUILD_ID`

Автоматическую регистрацию можно отключить переменной `AUTO_DEPLOY_COMMANDS=0`.
