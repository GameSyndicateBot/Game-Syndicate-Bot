# WORLD BOSS VISUAL V4

## Added
- Large 1600x900 Canvas battle board generated after every meaningful battle update.
- Boss card art, boss HP and rage bars, round and turn deadline.
- Team panel with player names, classes, HP, energy, shields, cooldowns and curse statuses.
- Current player highlight and compact initiative queue.
- Last battle actions shown directly on the board.
- "Моя карта" button with the real class card and current stats in an ephemeral response.
- "Мои призывы" button with summon HP and remaining rounds.
- "Враги" button with boss card, active boss minions and boss-specific summon list.
- "Журнал" button with the last 20 actions.
- Automatic text embed fallback if Canvas rendering fails.

## Preserved
- Manual testing channel and disabled automatic schedule.
- Existing World Boss balance, rage, AoE, curses, critical hits, summons and class mechanics.
