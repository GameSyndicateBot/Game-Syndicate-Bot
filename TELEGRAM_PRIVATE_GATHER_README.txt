TELEGRAM PRIVATE GATHER SETUP FIX

Что изменено:
- /gather можно вызвать в любом Telegram-чате;
- сообщение команды удаляется из исходного чата;
- меню создания открывается в личной переписке с ботом;
- готовый сбор публикуется только в назначенном Telegram game-lobby;
- сбор продолжает синхронизироваться с Discord;
- служебный чат «логи» больше не засоряется меню выбора.

Первичная настройка:
1. Добавь бота в Telegram-чат game-lobby.
2. Сделай бота администратором с правами:
   - отправлять сообщения;
   - удалять сообщения;
   - закреплять сообщения.
3. В game-lobby выполни:
   /setgatherchannel
4. Каждый организатор должен хотя бы один раз открыть личку с ботом и нажать Start.

После этого:
- /gather в любом чате → приватный мастер в личке;
- готовая созывалка → game-lobby + Discord.

Установка:
1. Ctrl + C
2. Распаковать ZIP в корень GS-Core с заменой.
3. node index.js

deploy-commands.js не нужен.
