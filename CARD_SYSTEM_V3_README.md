# Game Syndicate Card System V3

## Что добавлено
- 82 коллекционных варианта: 65 базовых (13 персонажей × 5 редкостей), 6 Exclusive, 10 Holographic и 1 Treasure.
- Один общий ежедневный пак. Раздельные типы паков не добавлялись.
- Реальные изображения выбираются по редкости карточки.
- Глобальные серийные номера для каждого сочетания `card_id + rarity`.
- Новые редкости: Exclusive, Holographic, Treasure.
- Учёт прогресса по вариантам, GS Score и повторок.
- Treasure защищена от распыления и может выпасть только один раз за календарный месяц на весь сервер.
- Магазин за Dust не выдаёт Treasure.

## Шансы обычного открытия
- Common — 50%
- Rare — 25%
- Epic — 14%
- Legendary — 8%
- Mythic — 2%
- Exclusive — 0.7%
- Holographic — 0.3%

Перед обычным броском отдельно проверяется Treasure: 0.02% (примерно 1/5000), только если в текущем месяце её ещё никто не получил.

## Карточки
Изображения лежат в `assets/cards/`.
Реестр находится в `data/cards.json`. Для новой карточки добавь запись с `drop_rarities` и `images`.

## После установки
1. Выполнить `npm install`.
2. Выполнить `npm run deploy` для обновления slash-команд.
3. Запустить `npm start`.
4. Проверить `/cardadmin give`, `/pack daily`, `/cards`, `/dust`, `/cardshop random`.

## Treasure
Получатель сохраняется в таблице `monthly_treasure_claims`. Поле `reward_status` по умолчанию `pending`; после выдачи награды его можно изменить вручную на `fulfilled`.
