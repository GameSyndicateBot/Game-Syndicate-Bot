PACK DAILY PUBLIC ACCESS FIX

Причина:
В commands/pack.js команда /pack daily была закрыта проверкой checkDeveloper().
Поэтому обычные участники не могли открыть ежедневный бесплатный пак.

Исправлено:
- /pack daily доступен всем участникам;
- кнопка Daily Pack в GS Hub продолжает работать;
- проверка "один пак в день" сохранена;
- административные команды не затронуты.

Установка:
1. Остановить бота: Ctrl + C.
2. Распаковать ZIP в корень GS-Core с заменой файла.
3. Запустить:
   node index.js

deploy-commands.js запускать не нужно.
