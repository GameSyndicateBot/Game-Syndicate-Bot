# GS World Boss

Мировой босс в отдельном канале в канале `1529226831797158130`.

## Расписание
- 13:00 МСК
- 20:00 МСК

## Ручной запуск
`/quickevent boss`

## Правила
- регистрация 10 минут;
- минимум 4 участника;
- гарантируется минимум 1 танк и 1 хиллер;
- d20 определяет инициативу;
- ход длится 60 секунд, затем выполняется автоатака;
- обычная атака даёт 20 энергии, промах 15;
- способность стоит 40, ульта 100;
- состояние боя хранится в SQLite и восстанавливается после перезапуска;
- обычные Quick Events не стартуют, пока активен мировой босс;
- после боя автоматически запускается следующий Quick Event.

## Основные файлы
- `services/worldBoss/config.js`
- `services/worldBoss/worldBossSystem.js`
- `commands/quickevent.js`
- `systems/quickEventSystem.js`
- `events/ready.js`
- `index.js`


## Тестовый режим
Автоматическое расписание сохранено в коде, но по умолчанию отключено. Запускать вручную: `/quickevent boss`. Для включения расписания установить `WORLD_BOSS_AUTO_SCHEDULE=true`.
