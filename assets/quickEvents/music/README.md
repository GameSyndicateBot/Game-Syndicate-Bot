# Музыкальные фрагменты Quick Event

Добавляйте сюда заранее подготовленные фрагменты длительностью 3–5 секунд в MP3/OGG.
После добавления файла укажите его в `manifest.json`:

```json
{
  "tracks": [
    {"file":"clip-01.mp3","title":"Believer","artist":"Imagine Dragons"}
  ]
}
```

Используйте только аудио, на которое у сообщества есть право использования.
Если валидных файлов нет, событие «Угадай музыку» не запускается и сообщает администратору, что база пуста.
