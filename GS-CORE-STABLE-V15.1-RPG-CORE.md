# GS CORE STABLE V15.1 — RPG CORE

Первый этап постоянной RPG-системы Game Syndicate.

## Добавлено

- `/hero create` — создание одного героя на участника.
- 12 классов, совместимых с концепцией World Boss.
- 10 происхождений с небольшими стартовыми бонусами.
- Уникальный последовательный номер героя и дата создания.
- Графическая карточка героя в фирменной фиолетово-чёрной стилистике GS.
- `/hero view`, `/hero stats`, `/hero equipment`, `/hero inventory`, `/hero history`.
- Справочники `/hero classes` и `/hero origins`.
- Уровень, XP, HP, сила, защита, ловкость, интеллект и удача.
- История героя.
- Подготовленные таблицы для экипировки, предметов, компаньонов, артефактов,
  репутации и экспедиций следующих этапов.

## После установки

1. Обновить файлы проекта.
2. Не заменять постоянную базу на хостинге вручную: таблицы создадутся автоматически.
3. Перерегистрировать команды: `npm run deploy:prod`.
4. Перезапустить бота.
5. Проверить `/hero classes`, затем создать тестового героя через `/hero create`.

## Ограничения этапа

Экспедиции и получение предметов будут включены в V15.2–V15.3. На этом этапе
инвентарь и экипировка уже работают как интерфейс и база, но изначально пусты.
