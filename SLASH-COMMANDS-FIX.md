# Исправление регистрации slash-команд

## Что изменено

- Удалена автоматическая регистрация команд при каждом запуске `index.js`.
- `deploy-commands.js` больше не удаляет серверные команды перед обновлением.
- Обычная регистрация выполняется одним безопасным `PUT`-запросом.
- Полное удаление оставлено только в отдельном явном режиме `npm run deploy:clean`.

## Как обновлять команды

На основном сервере выполнить один раз:

```bash
npm run deploy:prod
```

Или напрямую по ID сервера:

```bash
node deploy-commands.js 1493225843269439488
```

Не запускай регистрацию несколько раз подряд.

## Важно сейчас

Пока Discord возвращает ошибку `30034 Max number of daily application command creates has been reached`, повторная регистрация не пройдет. Новый архив предотвращает дальнейшее случайное удаление и повторную регистрацию при перезапусках.
