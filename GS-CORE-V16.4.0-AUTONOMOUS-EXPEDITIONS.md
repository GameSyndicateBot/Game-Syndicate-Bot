# GS CORE V16.4.0 — Autonomous Expeditions

- Six expedition tactics: balanced, cautious, aggressive, treasure, experience, resources.
- Tactics change risk and reward but never guarantee safety.
- Cautious heroes can still be ambushed, injured, fail, or lose Dust.
- Autonomous incidents are written into the final expedition report.
- Class selection is followed by tactic selection in Expedition Hub.
- Slash command `/expedition start` now requires a tactic.
- Resource, XP, Dust, rare find, success and injury modifiers are capped by the existing expedition outcome system.
