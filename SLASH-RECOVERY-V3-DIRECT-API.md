# Slash Recovery V3 — Direct API

- Убран запуск второго Node-процесса через `spawnSync`, который нестабильно работает на Bothost.
- Команды отправляются напрямую в Discord API из startup-скрипта.
- Запрос ограничен таймаутом 45 секунд.
- Ответ Discord, код 30034 и retry_after выводятся прямо в логи.
- Старые команды заранее не очищаются.
- Не более одной попытки в сутки до первого успешного восстановления.
