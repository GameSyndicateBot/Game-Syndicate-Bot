# Game Syndicate 3.0 — Commercial Ready, этап 1

## Новые команды

- `/setup` — мастер первого запуска.
- `/control` — панель настройки каналов без редактирования кода.
- `/diagnostics` — проверка Discord, Telegram, SQLite и настроенных каналов.
- `/status` — uptime, память, количество игроков/карт и размер базы.
- `/about` — карточка коммерческой сборки.

## Настройки в базе

Добавлена таблица `guild_settings`. Настройки сохраняются отдельно для каждого Discord-сервера.

Через `/control` настраиваются:

- логи;
- достижения;
- приветствия;
- Quick Event;
- Lucky Day;
- общие сборы Discord + Telegram;
- резервные копии.

В этой версии уже подключены к настройкам: серверные логи, достижения/level-up, приветствия, Lucky Day, Game Lobby и Discord-бэкапы. Quick Event отображается в панели и будет полностью переведён с прежних констант на следующем этапе.

## После обновления

1. Выполнить `npm install` или обычный деплой Bothost.
2. Выполнить `npm run deploy`, чтобы зарегистрировать новые slash-команды.
3. Запустить `/setup`.
4. Назначить каналы через `/control`.
5. Проверить `/diagnostics`.
