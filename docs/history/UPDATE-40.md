# GS Core Stable 2

Безопасное обновление для действующего Discord/Telegram-сервера.

## Исправлено

- Lucky Day больше не проводит розыгрыш и не выдаёт награду на сервере, где канал не настроен или недоступен.
- Выбор канала Lucky Day через `/setup` и `/lucky channel` сохраняется единообразно.
- Канал достижений из `/setup` теперь используется для обычных достижений и наград за категории, а не только для level-up.
- Убран шумный лог о повторно пропущенном level-up; защита от дублирования сохранена.
- Ручной запуск `/lucky run_now:true` теперь сообщает понятную причину, если канал не настроен или недоступен.

## Не менялось

- Игровая логика, награды и шансы.
- Quick Event и мировой босс внутри него.
- Discord/Telegram базы и существующие данные.
- Текущие персональные настройки рабочего сервера.
