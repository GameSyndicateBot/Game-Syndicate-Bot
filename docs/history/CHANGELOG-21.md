# Game Syndicate Bot — v21 Commercial Audit

## Исправлено

- Каналы Quick Event, загадок и Telegram GS Call получили настраиваемые переменные окружения с совместимыми fallback-значениями для текущего сервера.
- Исключаемые голосовые каналы теперь задаются списком `EXCLUDED_VOICE_CHANNEL_IDS`.
- Каналы правил и анонсов, а также Telegram-ссылка в приветствии стали настраиваемыми.
- Добавлена проверка формата Discord ID и публичных URL. Некорректные значения не приводят к падению запуска.
- Удалено автоматическое принудительное изменение статистики конкретного участника Quick Event. Одноразовое восстановление возможно только при явно заданном `QUICK_EVENT_SEED_USER_ID`.
- Добавлены команды `npm run check` и `npm run release:audit`.
- Коммерческий ZIP больше не содержит рабочие SQLite-базы, `.bak` и одноразовые скрипты награждения.

## Совместимость

Изменения не меняют структуру базы данных и не требуют миграции. Старые значения каналов сохранены как fallback там, где это необходимо для текущего сервера.

## Установка поверх работающего бота

1. Остановить процесс бота.
2. Распаковать обновление поверх текущей папки с заменой файлов.
3. Не удалять текущую папку перед распаковкой: `.env` и рабочие SQLite-базы остаются в ней.
4. Запустить `npm run preflight`, затем обычный запуск.
