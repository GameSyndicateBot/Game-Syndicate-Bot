# GS Build 39 — Safe live-server update

Обновление предназначено для установки поверх действующего Discord/Telegram-бота.

## Изменения
- Удалена отдельная незапущенная система загадок и её изображения.
- Загадки внутри Quick Event не изменялись.
- Мировой босс остаётся частью Quick Event.
- `/setup` переведён в режим настройки действующего сервера без сброса данных.
- Канал Quick Event, выбранный в `/setup`/`/control`, теперь применяется сразу и сохраняется.
- Удалены устаревшие дубли корневых Telegram-файлов, пустой обработчик события, `.bak` и одноразовый персональный Dust-скрипт.
- Docker cleanup исправлен для удаления старых файлов загадок при обновлении поверх предыдущих версий.
- Рабочие SQLite/WAL/SHM и `.env` не входят в update ZIP.

## Безопасность данных
Архив не содержит базы и не перезаписывает данные из `/app/shared`.
