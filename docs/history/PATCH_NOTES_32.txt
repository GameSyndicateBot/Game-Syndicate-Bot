Game Syndicate Bot 32 — GS registration persistence fix

Исправлено:
- Telegram GS-регистрация теперь хранится в постоянной БД (/app/shared по умолчанию).
- Выбранные через /gsregister эмодзи сохраняются после перезапуска и обновления бота.
- /gs упоминает только явно зарегистрированных участников.
- Бот больше не назначает случайные эмодзи обычным участникам автоматически.
- Старые ошибочные автоматические назначения очищаются при первой миграции, поэтому эмодзи больше не отображаются занятыми без причины.
- Добавлены эмодзи 😈 и 👿.

Важно:
Из-за старой ошибки невозможно надежно отличить реальные выборы участников от автоматически назначенных значений. После установки этой версии участникам нужно один раз заново вызвать /gsregister и выбрать эмодзи. После этого выбор будет сохраняться постоянно.
