ГОТОВЫЕ КАРТОЧКИ GAME SYNDICATE

Готовые 85 файлов находятся в:
output-final\000002 ... output-final\000019

Для повторной полной сборки:
npm run cards:final

Что делает новый сборщик:
- сохраняет исходные персонажей, фон и описания;
- берёт внешний верхний/боковой/нижний каркас из эталонов reference;
- применяет цвета Common/Rare/Epic/Legendary/Mythic;
- защищает нижний информационный блок от перекраски;
- исправляет номер карты и номер экземпляра;
- в Rare слово ЭКЗЕМПЛЯР и номер окрашены синим.

Важно:
output и output-reference были удалены как тестовые/неверные.
Для подключения к Daily Pack, альбому и распылению использовать только output-final.
