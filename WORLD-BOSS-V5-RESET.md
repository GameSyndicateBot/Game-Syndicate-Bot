# World Boss V5 — Boss Reset

Добавлена админ-подкоманда:

`/quickevent boss-reset`

Она:
- находит текущую регистрацию или активный бой;
- останавливает связанный таймер;
- переводит бой в состояние `cancelled`;
- очищает дедлайны;
- по возможности удаляет старое сообщение боя;
- позволяет сразу снова использовать `/quickevent boss`.

Также исправлено залипание при ошибке отправки сообщения:
- если Discord возвращает Missing Permissions или другую ошибку отправки;
- созданная запись автоматически отменяется;
- флаг занятости освобождается;
- следующий запуск не блокируется ложным активным боссом.
