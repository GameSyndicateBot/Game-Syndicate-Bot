# GS CORE STABLE V15.3 — ITEMS & EQUIPMENT

## Добавлено
- 45+ шаблонов предметов с редкостью, историей и бонусами.
- Слоты: оружие, броня, шлем, перчатки, ботинки, кольцо, амулет, рюкзак.
- `/hero inventory [category]`, `/hero item id`, `/hero equip id`, `/hero unequip slot`.
- `/hero collection` — постоянная энциклопедия найденных предметов.
- Характеристики героя и карточка учитывают экипировку.
- Экипировка влияет на шанс успеха экспедиций и шанс редкой добычи.
- Экспедиции выдают предметы разных уровней редкости в зависимости от сложности.
- Новый герой получает стартовый предмет по классу.

## Установка
1. Перед обновлением выполнить `/backup`.
2. Заменить файлы проекта архивом.
3. Выполнить `npm run deploy:prod`.
4. Перезапустить контейнер.
