LUCKY DAY — УСТАНОВКА

1. Скопируй файлы патча в корень проекта с заменой.
2. Зарегистрируй новую slash-команду:
   node deploy-commands.js
3. Запусти бота:
   node index.js
4. Один раз в Discord укажи канал публикации:
   /lucky channel:#нужный-канал

После этого бот ежедневно в 12:00 МСК проверяет вчерашние ежедневные задания.
Участвуют только игроки, которые выполнили И забрали награды за все задания.
Победитель предыдущего розыгрыша не может выиграть два дня подряд.

Награды:
55% — 150 Dust
25% — 300 Dust
12% — Base Pack (сразу открывается и выдаёт случайную карту)
6% — 600 Dust
2% — Premium Pack (сразу открывается и выдаёт случайную карту)

Команда /lucky показывает статистику.
Администратор может безопасно проверить запуск через /lucky run_now:true.
Повторный запуск за одну дату не выдаёт вторую награду.
