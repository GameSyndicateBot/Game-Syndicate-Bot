# GS CORE V16.3.0 — CLASS PROGRESSION

- У каждого героя отдельный уровень и XP для всех 12 классов.
- Перед экспедицией игрок выбирает класс; XP получает только выбранный класс.
- World Boss по-прежнему разрешает выбрать любой класс.
- Выбранный класс получает небольшие бонусы от своего уровня экспедиций: максимум +8% урона, +6% HP и +4% сопротивления на Lv.50.
- Общие бонусы героя и экипировки дополнительно ограничены, чтобы новичок оставался полезным.
- Старые Следопыт/Чернокнижник/Друид мигрируют в Лучник/Некромант/Бард.
