# GS Slash Recovery Auto Fix

При запуске контейнера бот автоматически пытается восстановить slash-команды на сервере `1493225843269439488`.

Защита от повторного лимита Discord:

- не более одной попытки в сутки;
- после успешной регистрации повторные deploy полностью прекращаются;
- при ошибке регистрации основной бот всё равно запускается;
- команды заранее не удаляются.

Маркеры хранятся в постоянной папке `/app/data`.
