GS TELEGRAM MOBILE CLIENT v1 — ДЛЯ GS-CORE(18)

Что включено:
- рабочее главное меню Telegram;
- безопасная привязка Telegram ↔ Discord;
- профиль GS из общей базы;
- баланс GS Dust;
- статистика коллекции по редкостям;
- последние достижения;
- настройки и отвязка Discord;
- единая таблица связи для профиля и будущих кроссплатформенных сборов.

Установка:
1. Остановить бота: Ctrl + C.
2. Распаковать ZIP в корень GS-Core с заменой файлов.
3. Зарегистрировать Discord-команду:
   node deploy-commands.js
4. Запустить:
   node index.js

Использование:
Telegram:
- /start или /menu
- /link
- /profile
- /dust
- /cards
- /achievements
- /settings

Привязка:
1. В Telegram нажать «Привязать Discord».
2. Скопировать шестизначный код.
3. В Discord выполнить /linktelegram code:XXXXXX.

Примечание:
Карты в этой версии показываются как статистика по редкостям.
