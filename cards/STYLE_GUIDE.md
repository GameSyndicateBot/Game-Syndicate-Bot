# GS Card Style Guide v1.0

Главное правило: карточка не генерируется заново. Она собирается движком из слоёв: арт + рамка редкости + UI + текст.

## Неизменные элементы
- арт персонажа;
- фон;
- композиция;
- описание;
- фиолетовые эффекты арта;
- фиолетовые внутренние линии и иконки, если они не являются частью рамки редкости.

## COMMON
Серебро, 1/5 звёзд. Нижний GS и номер экземпляра — фиолетовые.

## RARE
Тёмно-синий, 2/5 звёзд. Номер экземпляра может оставаться фиолетовым по эталону 000001.

## EPIC
Фиолетовый, 3/5 звёзд.

## LEGENDARY
Золотой, 4/5 звёзд. Рамка, имя, верх, GAME SYNDICATE, номер карты, полосы, STAFF/FOUNDERS, нижний GS, ЭКЗЕМПЛЯР и #000000 — золотые. Текст способности вроде ГАРМОНИЯ/СОЗЫВ остаётся серым, если это основной текст, а не элемент редкости.

## MYTHIC
Красный, 5/5 звёзд. Рамка, имя, верх, GAME SYNDICATE, номер карты, полосы, STAFF/FOUNDERS, нижний GS, ЭКЗЕМПЛЯР и #000000 — красные.
