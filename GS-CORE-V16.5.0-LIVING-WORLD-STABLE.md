# GS CORE V16.5.0 — Living World Stable

Добавлено ядро общего мира Discord-сервера:

- `/world map` — карта регионов, опасность, события, репутация и общий прогресс исследования;
- `/world region` — подробная карточка региона;
- `/world contracts` — ежедневные серверные контракты;
- постоянная таблица регионов и репутации для каждого Discord-сервера;
- ежедневная детерминированная смена региональных событий;
- общие контракты на экспедиции, победы, материалы и репутацию;
- экспедиции автоматически повышают репутацию региона и прогресс контрактов;
- новые регионы постепенно открываются от общего прогресса сообщества;
- все миграции добавочные и не удаляют существующие данные.

Версия package.json: 16.5.0.
