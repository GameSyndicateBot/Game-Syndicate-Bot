GS Telegram Ecosystem v1

Установка:
1. Распаковать в корень GS-Core с заменой.
2. Выполнить node deploy-commands.js
3. Запустить node index.js
4. В Discord в нужном канале выполнить /setgatherchannel
5. В Telegram открыть /start
6. Нажать «Привязать Discord», получить код и в Discord выполнить /linktelegram code:КОД

Возможности:
- главное Telegram-меню;
- общий профиль, Dust, карты и достижения;
- привязка Telegram ↔ Discord;
- общие сборы с единым списком участников на обеих платформах.
